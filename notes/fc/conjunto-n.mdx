# Definición

El conjunto $$\mathbb{N}$$ se define inductivamente como:

- $$0 \in \mathbb{N}$$
- Si $$n \in \mathbb{N}$$, entonces $$n+1 \in \mathbb{N}$$

El conjunto $$\mathbb{N}$$ cumple con el [Principio del Buen Orden](pbo).
